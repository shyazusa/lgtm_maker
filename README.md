# lgtm_maker

## Usage

### install sinatra

```
$ gem install sinatra
```

### setup lgtm_maker

```
$ git clone git@github.com:shyazusa/lgtm_maker.git
$ cd lgtm_maker
$ bundle install
$ mkdir public
$ mkdir public/images
$ mkdir public/fonts
```

#### Add fonts

Download and install from here.  
[もじワク研究 | マメロン オリジナル日本語フォント](https://moji-waku.com/mamelon/)

### Use!

```
$ bundle exec ruby app.rb
```

Accsess to `http://localhost:4567/`
